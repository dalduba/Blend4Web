Methods

auto_rotate(auto_rotate_ratio, callbackopt)

Switch auto-rotating of the TARGET or HOVER camera around its pivot, or auto-rotating of the EYE camera around itself. When it is called for the first time, auto-rotating is enabled while the next call will disable auto-rotating.
Parameters:
Name Type Attributes Description
auto_rotate_ratio Number Multiplier for the rotation speed
callback AutoRotateDisabledCallback <optional>
 Callback to be executed when auto-rotating is disabled

track_to_target(cam_obj, target, rot_speed, zoom_mult, zoom_time, zoom_delay, callbackopt, zoom_cbopt)

Smoothly rotate the EYE camera to make it pointing at the specified target (an object or some position). Then smoothly zoom on this target, pause and zoom back.
Parameters:
Name Type Attributes Description
cam_obj Object3D Camera object 3D
target Object3D | Vec3 Target object or target position
rot_speed Number Rotation speed, radians per second
zoom_mult Number Zoom level value
zoom_time Number Time it takes to zoom on the target, seconds
zoom_delay Number Delay before the camera zooms back, seconds
callback TrackToTargetCallback <optional>
 Finishing callback
zoom_cb TrackToTargetZoomCallback <optional>
 Zoom callback
